Tints.dev is a free palette generator and REST API that creates complete 11-stop color palettes (50–950) from a single hex color value. Designed for Tailwind CSS workflows, it outputs ready-to-use config for both Tailwind v3 and v4, with support for oklch, hsl, hex, and display-p3 color formats.

## What Tints.dev does

Starting from a single hex color, Tints.dev generates a full 11-stop palette spanning from near-white (50) to near-black (950) — the same range used by Tailwind CSS's built-in color palette. It uses either HSLuv (perceived) or standard HSL (linear) color math to distribute lightness evenly across stops.

## Get started

<Steps>
  <Step title="Open the generator">
    Go to [tints.dev](https://tints.dev) — a random palette loads automatically.
  </Step>
  <Step title="Enter your hex color">
    Type your brand color as a 6-character hex value (without `#`) into the color input field.
  </Step>
  <Step title="Adjust the palette">
    Use the hue, saturation, and lightness controls to fine-tune the output. Switch between perceived and linear color modes.
  </Step>
  <Step title="Copy your config">
    Select your Tailwind version (v3 or v4) and output format, then copy the generated config snippet into your project.
  </Step>
</Steps>

<Tip>
  You can also use the [REST API](/api/introduction) to fetch palettes programmatically — no API key required.
</Tip>
